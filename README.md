# 🎓 Behavioural Early Warning Intelligence System

An advanced behavioural analytics platform for early detection of student burnout and dropout risk using behavioural pattern modelling, predictive intelligence, and decision support strategies.

---

## 📌 Problem Statement

Universities often detect academic distress only after performance has declined.  
By that time, intervention becomes reactive rather than preventive.

This system proactively identifies behavioural signals such as:

- Reduced LMS activity
- Assignment submission delays
- Attendance changes
- Sentiment shifts
- Study pattern irregularities

To predict:

- Burnout Risk Level (Low / Moderate / High)
- Dropout Probability
- Academic Disengagement Indicators

---

## 🧠 Solution Overview

We designed a **Behavioural Decision Intelligence System**, not just a classification model.

The system includes:

- Advanced behavioural feature engineering
- Composite risk scoring (0–100 scale)
- Percentile-based dynamic risk segmentation
- Dropout probability modelling
- Behavioural trigger identification
- Structured intervention recommendation engine
- Institutional analytics dashboard

---

## 📊 Key Features

✔ Dynamic Risk Score (0–100)

## 🧮 Behavioural Feature Engineering

The system derives behavioural intelligence using engineered indicators:

- **Academic Stress Index**
- **Engagement Volatility Score**
- **Emotional Distress Index**
- **Study Pattern Variability Index**
- **Composite Behavioural Risk Score**

These engineered features improve interpretability and practical feasibility.

---

## 🤖 Modelling Approach

- Random Forest – Burnout Classification
- Logistic Regression – Dropout Probability
- KMeans – Behavioural Clustering
- Percentile-Based Segmentation – Risk Categorization
- OLS Trendline Analysis – Behaviour Relationships
